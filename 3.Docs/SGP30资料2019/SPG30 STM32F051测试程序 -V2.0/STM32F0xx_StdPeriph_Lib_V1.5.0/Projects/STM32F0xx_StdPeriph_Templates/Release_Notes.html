Release Notes for STM32F0xx Standard Peripherals Library Templates

Update History

V1.5.0 / 05-December-2014

Main Changes

  • Update all projects to add new workspace for STM32F070x6, STM32F070xB and STM32F030xC  devices.

V1.4.0 / 24-July-2014

Main Changes

  • Update all projects to add new workspace for STM32F091 devices.

V1.3.1 / 17-January-2014

Main Changes

  • Update TrueSTUDIO and RIDE7 project templates.

V1.3.0 / 16-January-2014

Main Changes

  • Update all projects to use new STM32F0xx device family names.
    • STM32F0XX_LD replaced by STM32F031
    • STM32F0XX_MD replaced by STM32F051
  • Update all projects to add new workspace for STM32F072, STM32F042 devices.

V1.2.0 / 22-November-2013

Main Changes

  • Update all projects to add new workspace for STM32F030 value line devices.

V1.1.0 / 31-July-2013

Main Changes

  • Update all projects to add new workspace for STM32F0xx Low-density devices, STM32F0XX_LD(STM32F0x0xx)
  • Template program simplified

V1.0.0 / 18-May-2012

Main Changes

  • First official release for STM32F0xx devices
