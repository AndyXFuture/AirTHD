Release Notes for STM32F0xx Standard Peripherals Library

Copyright © 2014 STMicroelectronics

 

This software is superseded by the STM32Cube solution, an all-in-one embedded software offer. ST continues its support for this software. However, for new designs with STM32 F0, it is recommended to take advantage of the fully integrated STM32CubeF0 solution, available at http://www.st.com/stm32cubef0
